# Smart Vehicle Monitoring and Driver Safety System

## Overview
A compact plug-and-play vehicle monitoring system designed to improve driver safety and fleet management using GPS, LoRa communication, and computer vision technologies.

## Problem Statement
Traditional vehicle monitoring systems often rely on internet connectivity and provide limited driver behavior analysis. This project addresses these limitations through intelligent monitoring and hybrid communication capabilities.

## Key Features
- GPS-based real-time vehicle tracking and trip logging
- Driver drowsiness detection using computer vision
- Geofencing and overspeed alert generation
- LoRa-based GSM-independent communication
- Offline SD card data storage
- Real-time dashboard monitoring
- Hybrid online/offline operation
